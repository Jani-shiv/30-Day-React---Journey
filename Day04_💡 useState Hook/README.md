## 🔥 Project Overview

Welcome to **Day 4** of my **30-Day React Journey**!  
Today’s project is a **Counter App** built using the `useState` hook in React.

This app will demonstrate how to manage the state in a functional component with `useState` and how to update the UI based on the state.

---

## 🖼️ App Preview

### Features:
- 🔢 Increment and decrement counter values
- 🚀 Interactive button controls
- 🎨 Sleek UI with a modern design
- ⚛️ Implementing `useState` for state management
